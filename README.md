# dotfiles

## Prerequire:
 - curl
 - emacs
 - git
 - go
 - python

```
$ sudo apt install curl emacs git golang python
```

## tmux
```
$ wget https://raw.githubusercontent.com/atetubou/dotfiles/master/.tmux.conf -O ~/.tmux.conf
```

## Install:
```
$ git clone https://github.com/atetubou/dotfiles.git
$ ./dotfiles/install.sh
$ source ~/.bashrc
```
